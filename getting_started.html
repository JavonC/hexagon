What is Hexagon

Hexagon is a microservices framework written in [Kotlin] which pursose is to ease the building of services (Web applications, APIs or queue consumers) that run inside cloud platforms.

It is meant to provide abstraction from underlying technoligies to be able to change them with minimum impact.

It only supports Kotlin use, Java is not a targeted language for the framework.

Why it was created

  • More time reading framework docs than coding
  • Frameworks targeted to everyone instead my custom needs
  • Easy to hack better than work ok for every use case
  • For fun!
  • To learn Kotlin
